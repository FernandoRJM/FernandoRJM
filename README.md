# Te cuento un poco sobre mí

Me especializo en la intersección entre la Ingeniería Comercial y la Ciencia de Datos, aplicando modelos avanzados de Machine Learning para optimizar operaciones logísticas, automatizar decisiones complejas y generar valor económico medible en industrias estratégicas.

---

## 🛠️ Tecnologías y Herramientas

Lenguajes de Programación: Python (Pandas, NumPy, Scikit-Learn, Joblib), SQL.
Desarrollo de Aplicaciones: Flet Framework (Construcción de aplicaciones ágiles e interactivas).
Visualización y Análisis (EDA): Matplotlib, Seaborn.
Integraciones: Consumo de APIs REST (Open-Meteo, datos meteorológicos en tiempo real).

## 🚢 Proyectos Destacados

Aquí puedes explorar mis proyectos de código abierto más recientes, enfocados en resolver desafíos logísticos reales en el sur de Chile:

1.  **[Análisis Predictivo Logístico: Optimización de Flotas](https://github.com/FernandoRJM/An-lisis-predictivo-log-stico-optimizaci-n-de-flotas-en-el-sur-de-Chile)**

2.  **[Alden: Sistema Predictivo de Riesgo Logístico Acuícola]([https://github.com/FernandoRJM/alden-riesgo-logistico](https://github.com/FernandoRJM/Alden-Sistema-Predictivo-de-Riesgo-Log-stico-Acu-cola))**
    ¿En qué consiste? Asistente interactivo inteligente que integra variables meteorológicas en tiempo real mediante APIs y aplica el algoritmo de seguridad adaptativo *"Windy-Adapt"* para estimar la viabilidad del zarpe de embarcaciones.
